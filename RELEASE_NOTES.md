# CodexPulse 0.3.0

This release makes CodexPulse easier to use on more Macs, especially if you want to switch languages or understand why live quota data is not available yet.

## What's New

- Added English, Simplified Chinese, and Traditional Chinese app text.
- Added a Language option in Preferences, so you can use System language or pick a language manually.
- Improved Chinese wording across menus, Preferences, alerts, empty states, and notifications.
- Clarified that CodexPulse works best with the official Codex app installed, opened, and signed in.
- Added clearer messages when Codex is missing, not ready, temporarily unavailable, or only cached data is available.

## Good To Know

CodexPulse reads rate-limit information from the local Codex app helper. It does not send your usage data anywhere.

For the most accurate live data, install the official Codex app in `/Applications`, open it once, and make sure you are signed in.

## First Launch Note

This build is ad-hoc signed, so macOS may ask for confirmation the first time you open it. If that happens, right-click `CodexPulse.app`, choose **Open**, then confirm.

Future releases can be Developer ID signed and notarized once signing credentials are available.
