# CodexPulse 0.2.0

Production polish release.

CodexPulse is still a tiny menu bar indicator, but it now has a proper Preferences window, clearer diagnostics, and optional notifications.

## Highlights

- Choose whether the top bar shows remaining usage, used usage, or both.
- Set refresh timing to 30 seconds, 60 seconds, or 5 minutes.
- Opt in to low-limit and stale-data notifications.
- Open Diagnostics for source, cache, last refresh, and last error details.
- Use Check for Updates to jump to the latest GitHub release.
- See friendlier messages when Codex is missing, not ready, cached, or temporarily unavailable.

## First Launch Note

This early build is ad-hoc signed, so macOS may ask for confirmation the first time you open it. If that happens, right-click `CodexPulse.app`, choose **Open**, then confirm.

Future releases can be Developer ID signed and notarized once signing credentials are available.
